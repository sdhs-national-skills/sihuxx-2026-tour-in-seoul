## `drawMap()` — 지도 그리기

```javascript
ctx.clearRect(...)  // 캔버스 전체 지우기
```

1. **라인** — 역마다 연결된 다음 역까지 선 그림 (색은 호선별)
2. **원** — 각 역 위치에 원 그림
3. **이름** — 역 이름 텍스트 표시

---

## `drawRoute(course)` — 선택한 경로 하이라이트

```javascript
drawMap()  // 지도 초기화 후 다시 그림
```

1. **환승역 파악** — 경로 중 이전 엣지랑 다음 엣지의 호선이 다른 역을 `transferSet`에 저장
2. **노란 라인** — 경로를 따라 굵은 노란선 그림
3. **마커** — 출발(파랑), 도착(빨강), 환승(주황), 일반(노랑) 원으로 표시

## `canvas.addEventListener('mousedown')` — 역 클릭

```javascript
// 클릭한 좌표랑 역 좌표 차이가 10 이내면 그 역으로 인식
if (xDiff <= radius && yDiff <= radius)
```

역 클릭 시 모달 띄우고, 출발지/도착지 버튼 클릭 시 해당 역 `idx` 저장

## `settingPlace()` — 출발/도착지 텍스트 업데이트

```javascript
$(".start-place").textContent = findStationByIdx(startPlace)?.name
```

저장된 `idx`로 역 이름 찾아서 화면에 표시

---

## `getDistance()` — 두 역 사이 거리 계산

```javascript
Math.round(Math.sqrt((x2-x1)² + (y2-y1)²))
```

중학교 수학 피타고라스 공식으로 좌표 간 거리 계산

---

## `map` 구성 — 그래프 만들기

```javascript
map[역idx] = [{ to: 다음역idx, line: 호선, distance: 거리 }, ...]
```

모든 역의 연결 정보를 `map` 객체에 저장. 나중에 경로 탐색할 때 씀.

---

## `getTop5Paths()` — 경로 탐색 (DFS)

출발역부터 시작해서 갈 수 있는 모든 경로를 탐색함

```javascript
if (visited.has(current)) return  // 이미 방문한 역이면 중단 (무한루프 방지)
if (current == endNode)           // 도착역 도달 시 경로 저장
```

찾은 모든 경로를 거리 짧은 순으로 정렬 후 상위 5개만 반환

---

## `renderCourseList()` — 코스 목록 렌더링

```javascript
const sorted = [...paths].sort(...)         // sort값에 따라 거리순/환승순 정렬
list.innerHTML = sorted.map(...).join("")   // 목록 HTML 생성
```

각 항목 클릭 시 → `selected` 클래스 추가 + `drawRoute()` 호출

## 버튼 이벤트들

```javascript
sortBtns.forEach(...)     // 클릭한 버튼 index(i)를 sort값으로 넘김 (0=거리, 1=환승)

// 일정설정 버튼

```
